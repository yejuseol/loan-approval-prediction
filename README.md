# Loan Approval Prediction

A machine learning project that predicts loan approval using applicant financial and credit information.

## Overview

This project analyzes a loan application dataset and builds classification models to predict whether a loan will be approved.  
The workflow includes data cleaning, feature encoding, feature selection, model training, and result evaluation.

## Key Features

- Data preprocessing and cleaning
- Feature encoding and selection
- Classification using Random Forest and Decision Tree
- Model comparison using accuracy
- Interpretation of important predictors

## Results

- **Random Forest accuracy:** 74%
- **Decision Tree accuracy:** 67%
- **Credit history** was identified as the strongest predictor of loan approval
